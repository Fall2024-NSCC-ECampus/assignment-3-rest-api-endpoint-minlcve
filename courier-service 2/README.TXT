# Courier Service Web Application

This is a simple web application designed to handle courier services, including user login, shipment tracking, and information about the company.

## Features

- **Home Page**: Introduction to the courier service and links to other sections.
- **Login Page**: A simple login form to access user accounts.
- **Track Shipment Page**: Allows users to track their shipments using a tracking number.
- **About Us Page**: Information about the courier company.

## Requirements

- A modern web browser (Chrome, Firefox, etc.)
- Internet connection (if hosted online)
